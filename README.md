# 企業接班人養成術

一款單機、離線可玩的網頁經營模擬遊戲。你扮演一位小型企業的總經理（接班人），用五年（60 個遊戲月）的時間處理內外部事件、做出決策，讓公司從小型成長為中型，最後成功上市為大型企業。

線上遊玩：https://gwanlinho.github.io/ceo-successor/

## 玩法

- 每個月會有數件來自各部門與外部單位的事件等你決策；處理完當月事件後「結束本月」進行結算，數據隨之變化，再進入下一個月。
- 隨時可打開**報表中心**（六分頁含趨勢圖）與**新聞情報**（含真偽難辨的小道消息）輔助決策。
- 當關鍵數據（營收、淨值、員工數、連續獲利、上市審查等）達標，公司規模就會升級。現金見底又借不到錢會倒閉；股東信心歸零會被撤換。
- 難度分簡單／普通／困難，影響事件數量、危機頻率、經濟環境與計分。

## 技術

- 純原生 HTML / CSS / JavaScript（ES Modules），無框架、無 build 工具、無執行期依賴。
- 純函式引擎 + 單一 JSON 狀態樹 + 種子化亂數（可確定性重播、存讀檔一致）。
- 遊戲資料全部外部化為 `data/*.json`（改資料即改內容與難度，不需動程式）。
- PWA：安裝後完全離線可玩；localStorage 自動存檔 + 匯出／匯入。

## 本機執行

需以 http 伺服器開啟（fetch JSON 不支援 file://）：

```
python3 -m http.server 8000
# 瀏覽器開 http://localhost:8000
```

## 開發與測試

- 資料驗證：`node tools/validate-data.mjs`
- 引擎單元測試：`node tests/engine.test.js`
- 平衡模擬：`node tests/simulate.js [局數]`
- 端到端測試：`node tests/e2e-m4.mjs`、`node tests/e2e-pwa.mjs`（需先起 http.server 並設 PORT）

設計與開發文件見 `docs/`（企劃、架構、難度設計、開發計畫、測試日誌）。
